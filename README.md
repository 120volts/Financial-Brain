# Financial Brain Event Engine v0.1

Upload `index.html` to a separate GitHub Pages repository.

Tap **Load demo events** to see:
- a paid invoice arrive already understood,
- a Sony camera purchase needing two facts,
- an unknown bank deposit,
- and a W-2 document.

Open the camera purchase, answer the questions, and watch Tax Readiness rise.

This is an interaction prototype, not filing-grade tax software. Data stays in that browser/device.
