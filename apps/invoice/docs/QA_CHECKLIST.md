# Regression Checklist

## Navigation
- All five bottom tabs open.
- Selected tab is visibly highlighted.
- Refresh restores the previous tab.
- Refresh on New opens a fresh usable invoice.

## Invoice
- Create and save Draft.
- Preview invoice.
- Send/open email workflow.
- Mark Viewed.
- Mark Paid.
- Duplicate invoice.
- Edit project, dates, notes and lines.
- Delete confirmation works.

## Client and items
- Create/edit/archive/delete client.
- New invoice for client works.
- Create/edit/duplicate/archive/delete saved item.

## Documents
- PDF builds.
- Printable invoice opens.
- Share path works where supported.

## Persistence
- Existing localStorage data loads.
- Data remains after refresh.
- Brain events rebuild correctly.
- Paid tax records remain consistent.

## PWA
- Manifest loads.
- Service worker registers over localhost/HTTPS.
- Application shell opens after an offline reload.
