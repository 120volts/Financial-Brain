# Invoice App v2.3.4 — New Tab Refresh Fixed

## Fixed

Pull-down refresh now restores all five bottom navigation tabs:

- Invoices
- New
- Clients
- Services & Products
- Brain

When the last tab was **New**, the app reopens a fresh, usable New Invoice screen instead of jumping to Invoices or Clients.

A small `v2.3.4` marker is visible in the app so you can confirm the updated index is running.
